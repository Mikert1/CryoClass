What is CryoClass?

With help of CryoClass you can add styling quickly using short name classes or their fullname.
When i use frameworks i always feel like my freedom and creativity is taken away from me.
Allow me to show you an example:

Lets first add a Width

Bootstrap

w-25 - sets 25% width

sadly thats it with 25 50 75 and 100 in %

Tailwind

You can add width in pixels from gaps 2px-4px between each class.
And the max you can set is 384px.
For example:

w-1 sets 0.25rem; /* 4px */
w-1.5 sets 0.375rem; /* 6px */

You can add some most used % like anything in the table of 12 (5/12).
For example:

w-1/2 sets 50%;
w-1/3 sets 33.333333%;

And max and min width.

w-min sets min-content;
w-max sets 100%;

Even covered view width.

w-screen sets 100vw;

Tailwind does cover a lot thats nice

CryoClass (This framework)

You can add width in pixels with no limitations.
Standard px is chosen but you can say rem or em or anything that css normally supports.
For example:

w-300px sets 300px;
w-24rem sets 24rem;

You can add any % even 33.3333%
For example:

w-50% sets 50%;
w-68% sets 68%;

View width, you might even use calc in there.
If you can use it in css its here.
For example:

w-100vh sets 100vh;
w-calc(100vw/8+10px) sets calc(100vw / 8 + 10px);

Realy anything you can think of.
